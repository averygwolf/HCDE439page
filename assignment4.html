Assignment 4!

For this assignment I utilized the LCD library to display the number of times a button has been pressed.

Approach

For this assignment, I created a circuit that includes a button and an LCD display. Everytime the button is pressed, the button counter increases. The LCD displays the button counter and increases everytime the button is pressed again.

Here is the schematic for my libraries setup. I used a 10k potentiometer for the LCD display and a 10k ohm resistor for the button. I also included a 220ohm resistor for the LCD display, but it is unecessary and was included when I thought I'd utilize an LED.
